2. Getting Started

2.1 Recommended installation of ArcGIS Pro and extensions

  1. ArcGIS Pro Version: 3.3.2 or higher.

  2. ArcGIS Pro extensions: 3D Analyst, Data Interoperability, Data Reviewer, Geostatistical Analyst, Image Analyst, Publisher, Spatial Analyst.

  3. ArcGIS Online Account: ArcGIS online

2.2 Recommended Installation of open-source software

  1. Text editor: Notepad++

  2. Python IDE: Spyder

  3. Processing 3D building data: 3dcitydb and pgAdmin

  4. Processing LiDAR data: LAStools

2.3 Required Data

  1. 3D building data: Geoportal NRW

  2. LiDAR data: Geoportal NRW

  3. Tree cadastre: Open-data Dortmund

Note

Download and install all requirements to proceed. The installation and setup of ArcGIS Pro and its extensions is not part of this documentation. Everything else is covered in the following steps.

2.4 Installing the ArcGIS Pro extensions

  1. Open ArcGIS Pro

  2. In the left pane, navigate to –> Licensing

  3. Klick “Manage your data licensing”

  4. In the context menu “ArcGIS Pro Extensions” select the following extensions

    • 3D Analyst –> Provides tools for 3D feature and surface analysis.

    • Data Interoperability –> Is an integrated spatial toolset (ETL, extract, transform and load) that runs within the groprocessing framework using Safe Software’s FME technology. It enables you to integrate data from multiple sources and formats, use that data with geoprocessing tools, and publish it. Important for the integration of the 3D building dataset.

    • Data Reviewer –> Provides a variety of tools to help perfom and manage QA/QC of spatial data.

    • Publisher –> Enables the sharing of data throughout the ArcGIS Platform. Important for sharing your data in the ArcGIS online account.

    • Spatial Analyst –> Provides advanced spatial modeling and analysis tools for raster data. Important for the analysis of raster layers.

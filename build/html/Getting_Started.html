2. Getting Started

2.1 Recommended installation of ArcGIS Pro and extensions.

  1. ArcGIS Pro Version: 3.3.2

  2. ArcGIS Pro extensions: 3D Analyst, Data Interoperability, Data Reviewer, Geostatistical Analyst, Image Analyst, Publisher, Spatial Analyst.

  3. ArcGIS Online Account: ArcGIS online

2.2 Recommended Installation of open-source software.

  1. Text editor: Notepad++

  2. Python IDE: Spyder

  3. Processing 3D building data: 3dcitydb and pgAdmin

  4. Processing LiDAR data: LAStools

2.3 Required Data.

  1. 3D building data: Geoportal NRW

  2. LiDAR data: Geoportal NRW

  3. Tree cadastre: Open-data Dortmund
